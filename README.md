# Group-10
Micro Task Planner
1. Project Overview

Micro Task Planner is a web-based application designed to improve productivity by allowing users to break large tasks into smaller, manageable steps. The system promotes structured task completion and helps reduce procrastination by encouraging incremental progress.

2. Objectives
To reduce procrastination by simplifying task execution
To improve productivity through structured task management
To provide a simple and efficient task organization system
To enable users to track progress effectively
3. Features
Create and manage main tasks
Add subtasks or mini steps under each task
Mark individual steps as completed
Automatic calculation of task progress
Progress visualization using a progress bar
Edit and delete tasks and steps
4. Technologies Used
HTML – Structure of the application
CSS – Styling and layout design
PHP – Backend logic and functionality
MySQL – Database management
6. Database Structure
Tasks Table
id
title
created_at
Steps Table
id
task_id
step_text
is_completed
8. System Workflow
User creates a task
Task is divided into smaller steps
User completes steps one by one
System updates progress automatically

9. Example
Task: Study Mathematics
Open textbook
Read assigned pages
Solve practice problems

8. Benefits

Helps users start tasks easily
Reduces stress and workload perception
Improves focus and time management
Provides clear progress tracking

9. Future Enhancements

User authentication system
Reminder and notification feature
Deadline and priority management
Mobile-responsive interface

10. Installation and Setup

Install XAMPP or WAMP server
Copy project folder into htdocs directory
Import database into phpMyAdmin
Start Apache and MySQL services
Open browser and run: http://localhost/project-folder
